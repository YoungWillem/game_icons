# CHANGELOG

This gem follows [semantic versioning](http://semver.org), with the date of the latest game-icons.net release in the version number.

e.g. "0.1.0.20150413.alpha" has the April 13, 2015 release of game-icons.net

## v0.2.0 / Unreleased
